# Blinkit Dashboard

## Overview

Created a Power BI dashboard to analyze Blinkit sales performance and product trends using interactive visualizations.

## Tools Used

* Power BI
* Power Query
* DAX

## Features

* Interactive filters and slicers
* Sales analysis
* Product category analysis
* KPI cards
* Outlet performance analysis

* Dashboard Development
